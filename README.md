
# Conversor de Real para Dólar

Um conversor simples e moderno que permite transformar valores em Reais (BRL) para Dólares Americanos (USD) com a cotação atualizada em tempo real.

## Funcionalidades

- Entrada de valor em reais pelo usuário.
- Consulta automática da cotação atual do dólar via API pública [AwesomeAPI](https://economia.awesomeapi.com.br).
- Conversão instantânea e exibição do valor convertido com duas casas decimais.
- Interface amigável e responsiva, alinhada no centro da página.
- Mensagens de erro para valores inválidos ou problemas na consulta da API.

## Tecnologias utilizadas

- HTML5
- CSS3 (estilos internos para simplicidade)
- JavaScript moderno com `fetch` e `async/await`

## Como usar

1. Abra o arquivo `index.html` em seu navegador preferido.
2. Digite o valor em reais que deseja converter.
3. Clique no botão **Converter**.
4. Veja o valor convertido em dólares com a cotação atual.
